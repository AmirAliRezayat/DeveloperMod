# 🚀 DevMod

DevMod is a modern and customizable Chrome New Tab extension designed for developers.

DevMod یک داشبورد مدرن و شخصی‌سازی‌شده برای برنامه‌نویسان است که صفحه تب جدید کروم را به یک محیط زیبا و کاربردی تبدیل می‌کند.

---

## ✨ Features | امکانات

- 👋 Personalized greeting with your name
- ⏰ Live clock and Persian date
- 🔍 Google Search
- 🌐 Public IP display
- 📶 Internet connection status
- 🖼️ Random programming wallpapers
- 💎 Glassmorphism UI
- 🔤 Dana Persian font support
- 🚀 Quick access to developer tools

### Quick Access

---

## 📦 Installation

1. Download or clone this repository.
2. Open `chrome://extensions`.
3. Enable **Developer Mode**.
4. Click **Load unpacked**.
5. Select the DevMod folder.
